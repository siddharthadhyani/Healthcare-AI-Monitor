# 🏥 AI Healthcare Monitor

An AI-powered healthcare monitoring application built with Python and Streamlit. The application provides health insights, AI-assisted medical guidance, voice input support, and interactive health tracking dashboards.

## 🚀 Live Demo

Live App: https://healthcare-ai-monitor-3dhs4fzvlappppauqowrjevr.streamlit.app

## 📌 Features

* AI-powered healthcare assistant
* Voice input support
* Health monitoring dashboard
* Interactive charts and visualizations
* Patient profile management
* Groq AI integration
* Responsive Streamlit interface

## 🛠️ Tech Stack

* Python

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/siddharthadhyani/Healthcare-AI-Monitor.git
cd Healthcare-AI-Monitor
```

### Create Virtual Environment

```bash
python3 -m venv venv
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Environment Variables

Create a `.env` file:

```env
GROQ_API_KEY=YOUR_API_KEY
```

### Run Application

```bash
streamlit run ai2.py
```
